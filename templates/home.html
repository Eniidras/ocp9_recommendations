Utilisation du modèle via l'api

Modèle d'utilisation :


# A curl example
curl -X POST https://APP_ADDRESS/api \
    -H 'Content-type: application/json' \
    -d '{"user_id" : ID_USER}'

Possibilité d'ajouter le paramètre filter_liked_items afin de ne recommander que des articles nouveaux. (par défaut false)

Ainsi que number_items pour changer le nombre d'articles recommandés. (par défaut 5)

Exemple :


curl -X POST http://APP_ADDRESS/api \
    -H 'Content-type: application/json' \
    -d '{"user_id" : ID_USER, \
        "filter_liked_items" : true, \
        "number_items" : 10}'
